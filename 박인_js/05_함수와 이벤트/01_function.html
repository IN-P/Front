함수란?

  • 함수는 어떤 일련의 작업을 수행하는 여러 명령어의 집합
  • 프로그램 상에서 자주 사용하는 기능을 함수로 만들어 사용하면 같은 코드 내용을 여러 번 작성할 필요가 없다.
  • 사용하는 기능을 함수로 만들어 사용하면 프로그램 정리가 간결하고 기능별로 정리할 수 있다.

함수 선언 및 호출

함수 선언
예약어 function을 사용하고 { } 안에 실행할 명령어를 작성
함수는 선언하는 것만으로는 실행되지 않고 호출해야 한다.
함수 호출(함수 실행)
함수명()
함수 선언과 함수 호출 순서는 상관없다.
함수 선언할 때 함수명은 식별자의 규칙에 맞추어 작성한다.

// 함수 선언 function 함수명() { 명령; } // 함수 호출 함수명();

매개변수와 인수

매개변수(parameter)
함수 선언 시 외부에서 값을 받아 줄 변수
매개변수는 선언된 함수 안에서만 사용한다.
매개 변수를 여러 개 사용할 때는 매개변수 이름 사이에 쉼표(,)를 찍어 나열한다.
매개 변수에 기본값을 지정할 수 있다.
인수(argument)
매개변수가 있는 함수를 호출할 때 함수로 넘기는 실제 값

return

  • return 문은 함수 실행의 결과값을 함수 호출 위치로 반환
  • return을 만나면 함수는 즉시 종료
  • 반환값을 변수에 저장하여 다른 연산에 활용할 수 있다.
  • return 문이 없는 함수는 보통 화면에 값을 출력하거나 특정 동작을 수행하는 데 사용된다.
  • return을 만나는 순간 함수는 즉시 종료된다.
